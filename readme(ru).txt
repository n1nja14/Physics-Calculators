Проект на тему "Калькулятор по физике". Этот калькулятор призван помогать в проверке домашнего задания при решении сложных задач.
В нём представлены: калькулятор, тематические блоки, такие как: "Электродинамика" и "Механика", в которых можно найти необходимые темы и 
два раздела с тестами для проверки знаний по вышеперечисленным блокам.
Ознакомиться с полным списком тем можно во вкладке “О программе”, на главной странице и по нажатию на выбранную тему перейти к ней.
Также добавлена функция перехода на предыдущую страницу(кнопка Esc) и вычислить на кнопку "Enter", для удобства и ускорения действий пользователя.

Калькулятор:
В данном разделе можно вписать выражение типа “(2-1*2)^3/1,5” и получить ответ.
Рекомендуем использовать запятую, а не точку, корень можно реализовать через (^0,5), а также прописывать все операция,
которые вы хотите совершить.

Любой из разделов “Электродинамика” и “Механика”:
В этих разделах существует функция подсчёта любой неизвестной, например, у нас есть задача на тему “Закон Джоуля-Ленца” и,
допустим, у нас есть все переменные, кроме количества теплоты. Нам необходимо просто вписать известные нам значения(все кроме
одной, которая нам необходима) и нажать на кнопку “Вычислить”. В пустом окне высветится необходимое значение.
Для запуска калькулятора необходимо открыть файл "Калькулятор по физике" и начать им пользоваться.
